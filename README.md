# StageTwo

StageTwo is the step to retrain the model and to see the performance change after generating new datasets using CycleGAN.

Workflow: 

1. Install [ml-spm](https://github.com/SINGROUP/ml-spm/tree/main) 

2. Train posnet and graphnet using original dataset, style translated dataset and augmented dataset (Original + Style translated). 

3. Test trained models using labeled dataset and experimental dataset.
